Creating a Batch (.bat) File to Run a Java Program  (Sample Program and Batch File)

The following commands placed in a text file saved as "compile_run_Monogram.bat" will compile and run the program Monogram.java.
set PROGRAM=Monogram
echo "compiling program %PROGRAM%.java ..."

javac %program%.java

java %program%

cmd.exe

The commands in this bat file assume that SDK has been installed and the the path variable is set.
